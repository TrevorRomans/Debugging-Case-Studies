# Debugging Case Studies

Welcome to my first official project in my pursuit of a debugging career! This repository will include a selection of projects I have previously worked on, and I will be going through the process of debugging them each. 

## Case 1: Custom Quicksort

When first learning sorting algorithms in my first coding class, we were given an online resource that visually simulated what the algorithms looked like when sorting lists of numbers. Since quicksort was shown but not covered code-wise in class, I chose in my free time to try and reverse engineer it based on the visual. Not only was my solution unrefined and longer than needed, but it was affected by a bug that took 3 days of work to resolve.
